Fundamentos de Deep Learning
Este repositorio agrupa mis proyectos iniciales en el campo del Deep Learning. Estos cuadernos representan la base técnica necesaria para comprender cómo funcionan las arquitecturas neuronales, desde la implementación de redes simples hasta redes convolucionales aplicadas a la visión artificial.

 Contenido del Módulo
Implementación ANN: Construcción de Redes Neuronales Artificiales desde cero, analizando la propagación y retropropagación (backpropagation) para entender la optimización de pesos.

CNNs con MNIST: Implementación de Redes Neuronales Convolucionales para la clasificación de dígitos escritos a mano. Este es el punto de partida fundamental en el estudio de la visión artificial.

CNNs con Sign Language Digits: Aplicación de arquitecturas convolucionales para reconocer lenguaje de señas, demostrando la capacidad de las CNN para extraer patrones espaciales en imágenes.

 Tecnologías
Python

TensorFlow / Keras

Redes Neuronales Artificiales (ANN)

Redes Neuronales Convolucionales (CNN)

 Valor del Proyecto
Estos ejercicios son fundamentales para demostrar que comprendo el "porqué" detrás de los modelos de IA. Antes de escalar a sistemas generativos avanzados, es vital dominar cómo una red aprende a clasificar datos estructurados e imágenes.
