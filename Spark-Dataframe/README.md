#  Procesamiento de Big Data con PySpark

Este proyecto documenta un flujo de trabajo de ingeniería de datos utilizando **Apache Spark (PySpark)**. El objetivo es realizar procesos ETL (*Extract, Transform, Load*) sobre grandes volúmenes de datos, demostrando capacidad de escalabilidad y manejo de DataFrames distribuidos.

##  Tecnologías Utilizadas
* **Python**
* **PySpark** (Spark DataFrames)
* **SQL** (para consultas analíticas sobre los datos)

##  Metodología de Trabajo
El cuaderno implementa las fases clave de un pipeline de Big Data:
* **Ingesta:** Carga de datasets masivos desde fuentes externas.
* **Limpieza:** Tratamiento de valores nulos, eliminación de duplicados y tipado de datos en un entorno distribuido.
* **Transformación:** Uso de funciones de ventana (*window functions*), agregaciones y transformaciones lógicas para extraer insights de valor.
* **Optimización:** Ejecución de consultas eficientes aprovechando la arquitectura de procesamiento en memoria de Spark.

##  Valor del Proyecto
Spark es la herramienta estándar en la industria para el procesamiento de datos a gran escala. Este proyecto certifica que soy capaz de transformar datos crudos y desordenados en información estructurada y lista para ser consumida por modelos de Machine Learning o tableros de BI.
