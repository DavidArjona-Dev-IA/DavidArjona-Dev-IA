Human Activity Recognition (UCI HAR)
Este proyecto implementa un sistema de clasificación para identificar actividades humanas (como caminar, subir escaleras o estar sentado) basándose en los datos capturados por los sensores (acelerómetro y giroscopio) de un smartphone.

 Tecnologías Utilizadas
Python

NumPy / Pandas (Manipulación de datos)

Scikit-Learn (Modelos de Clasificación)

Matplotlib / Seaborn (Análisis exploratorio)

 Pipeline de Datos
El proyecto sigue el ciclo completo de procesamiento de datos de sensores:

Ingesta: Carga y estructuración de datos de series temporales.

Preprocesamiento: Normalización y limpieza de las señales de los sensores.

Extracción de Características: Cálculo de métricas estadísticas (media, desviación estándar, etc.) sobre las ventanas temporales.

Modelado: Entrenamiento de modelos de clasificación para predecir la actividad con alta precisión.

 Valor del Proyecto
Este trabajo demuestra mi capacidad para manejar datos de sensores en bruto, transformarlos en características útiles y aplicar algoritmos de Machine Learning para obtener predicciones precisas. Es una habilidad fundamental para cualquier proyecto de Wearables o salud digital.

 Próximos pasos
Deep Learning: Implementar una Red Neuronal Recurrente (RNN/LSTM) para capturar las dependencias temporales de los sensores.

Análisis en Tiempo Real: Optimizar el pipeline para procesar flujos de datos en tiempo real.
